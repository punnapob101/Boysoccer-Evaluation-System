# Boy Soccer Skill Assessment

This application was developed specifically for the **Boy Soccer Academy** to provide an easy-to-use platform for assessing player skills and managing related data. 

Additionally, this project serves as a practical learning experience and practice project for building modern web applications using React and Vite.

## 🚀 Features

- **User-Friendly Interface**: Designed to be simple, clean, and highly intuitive for the staff at Boy Soccer Academy.
- **Skill Evaluation**: Streamlines the process of recording and tracking soccer skill assessments.
- **Responsive Design**: Works perfectly across various devices.

## 🛠️ Tech Stack

- **React** 
- **Vite**
- **Tailwind CSS** (for styling)

## 💻 Getting Started

To run this project locally on your machine, follow these steps:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. Open your browser and visit the local link provided in the terminal (usually `http://localhost:5173`).
